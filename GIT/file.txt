## GIT COMMANDS ##

1. Clone a project
git clone <project_url>

2. Check the status of change 
git status

3. Add files to the git
git add .

4 Add commit message
git commit -m <"commit message">

5. Push the change to github
git push -u origin <brnach_name>

6. Create a branch and checkout to branch
git checkout -b <branch_name_to_created>

7. To list the branches
git branch

8. Checkout to an existing branch
git checkout <branch_name>
